## Commands
  - avatar
    - 取得用戶頭像
  - help
    - 取得指令的說明
  - kick
    - 於伺服器踢出使用者 (_尚無功能_)
  - ping
    - Pong
  - prune
    - 批量刪除訊息
  - react
    - 只是一個偵測使用者回應的功能
  - reload
    - 重新加載命令
  - restart
    - 重新開始bot (僅限使用pm2執行時有作用)
  - say
    - 以機器人身分傳送訊息
  - server
    - 取得伺服器資訊
  - verification
    - 傳送伺服器驗證訊息
